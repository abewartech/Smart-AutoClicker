# Main Screen

The main screen is your command center. From here, you manage your configurations and start the automation service.

## Core Elements

### 1. Mode Selector
Toggle between **Smart Mode** and **Simple Mode**.
-   **Smart Mode**: Event-based automation with image detection.
-   **Simple Mode**: Standard auto-clicker with timed intervals.

### 2. Scenario List
A list of your saved automations.
-   **Create New**: Tap the FAB (+) to create a new scenario.
-   **Run**: Tap the Play icon next to a scenario to load it and show the overlay.

### 3. Service Status
The banner at the top shows if the **Accessibility Service** is running.
-   If Red: Tap it to open Android Accessibility Settings and enable "Klick'r".
